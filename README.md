# QQ群管插件 v3.1

基于 AstrBot 框架开发的QQ群管理插件，支持分群模式、分群管理员、数学/随机验证、主人管理等功能。

## ✨ 功能特性

### 📝 审核系统
- 数学验证（自动生成数学题，验证失败可踢出）
- 随机ID验证（生成验证码，用户输入验证）
- 管理员手动审核
- 自动审核（可选）
- 主人/群管理员邀请免验证

### 🚫 黑名单管理
- 用户黑名单（支持临时封禁和永久封禁）
- 关键词黑名单（自动检测敏感内容）
- 黑名单导入/导出功能

### 🔇 禁言管理
- 单人禁言（支持秒、分、时、天）
- 永久禁言
- 全体禁言/解禁
- 禁言列表管理

### 👢 踢出管理
- 普通踢出
- 踢出并拉黑
- 连带踢出（踢出用户及其邀请的所有成员）
- 踢出时自动撤回用户消息

### 🗑️ 撤回功能
- 按消息ID撤回
- 按用户撤回（带数量限制）
- 撤回用户最近消息
- 踢出时自动撤回用户消息

### 📢 欢迎功能
- 自定义欢迎语
- 固定前缀格式：欢迎 [用户名]进入[群名], 自定义内容
- 支持变量：{user}, {group}, {time}

### 💬 自动回复
- 支持按触发词回复
- 支持随机回复（用|分隔）
- 回复冷却时间
- 支持变量：{user}, {group}, {time}
- 可设置优先级
- 快捷指令：/回复 问 <问题> 答 <答案>

### 🏷️ 名片管理
- 修改群成员群名片
- 获取群名片信息

### 👤 用户互动
- 每日签到（连续签到奖励）
- 积分排行榜
- 签到排行榜

### 🔒 权限系统
- **机器人主人**：不受任何权限限制，可在所有群使用所有命令
- **群管理员/群主**：可使用群内所有管理命令、管理分群管理员
- **分群管理员**：可使用群管理功能（禁言、踢出、撤回、黑名单等）
- **普通用户**：仅可使用签到、排行榜、我的信息等普通命令

## ⚙️ 配置项

### 全局配置项（通过 /全局设置 设置）

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `owner_ids` | 机器人主人QQ列表 | `[]` |
| `enable_group_settings` | 是否启用分群配置 | `false` |
| `enable_global_settings` | 是否启用全局设置 | `true` |
| `welcome_enabled` | 是否开启欢迎功能 | `true` |
| `auto_approve` | 是否自动审核 | `false` |
| `anti_spam_enabled` | 是否开启反垃圾 | `true` |
| `sign_enabled` | 是否开启签到 | `true` |
| `auto_reply_enabled` | 是否开启自动回复 | `false` |
| `default_mute_duration` | 默认禁言时长(分钟) | `30` |
| `default_kick_reason` | 默认踢出理由 | `违规发言` |
| `spam_threshold` | 垃圾消息阈值 | `5` |
| `spam_time_window` | 垃圾消息时间窗口(秒) | `60` |
| `max_recall_count` | 单次最大撤回数量 | `10` |
| `admin_roles` | 管理员角色列表 | `['群主', '管理员']` |
| `allow_self_recall` | 是否允许用户撤回自己消息 | `false` |
| `allow_self_reply` | 是否允许用户使用回复功能 | `false` |

### 分群配置项（通过 /设置 设置）

分群配置与全局配置相同，新增：
- `custom_admins` - 分群自定义管理员列表

### 设置主人

## 🚀 安装说明

1. 将插件文件夹放入 AstrBot 的 plugins 目录
2. 重启 AstrBot 服务
3. 设置机器人主人（可选但推荐）：
   ```
   /添加主人 你的QQ
   ```
4. 在群内使用 `/群管帮助` 查看命令列表

## 🔒 权限说明

| 用户类型 | 权限范围 |
|----------|----------|
| 机器人主人 | 所有命令（所有群） |
| 群主/群管理员 | 群内所有命令（可管理分群管理员） |
| 分群管理员 | 群管理功能（禁言、踢出、撤回等） |
| 普通成员 | 仅普通命令 |
